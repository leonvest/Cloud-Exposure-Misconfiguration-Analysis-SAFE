# Cloud-Exposure-Misconfiguration-Analysis-SAFE

This repository contains **read-only** scripts to help you audit a cloud VM and
produce a structured findings report.

✅ Includes:
- host inventory
- listening ports snapshot
- firewall rule snapshot
- lightweight log triage
- metadata endpoint safety check (non-exploitative)
- reporting tools

❌ Not included:
- exploitation steps
- payloads
- privilege escalation instructions
- mass scanning scripts

Use only on systems you own or have explicit permission to assess.
